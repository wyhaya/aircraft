

# aircraft

A simple flying shooting game made with the [bevy](https://github.com/bevyengine/bevy) engine

## Play

```
cargo run
```

### Moves

<kbd>↑</kbd>
<kbd>↓</kbd>
<kbd>←</kbd>
<kbd>→</kbd>

### Shoot

<kbd>A</kbd> <kbd>S</kbd>
